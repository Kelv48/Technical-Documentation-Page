Introduction

Flask is a lightweight WSGI web application framework in Python. It is designed with simplicity and flexibility in mind.

It allows developers to quickly build web applications without extensive configuration.

from flask import Flask

With Flask, you can build anything from a simple "Hello, World!" app to complex web applications.

Setup

To get started with Flask, you'll need to install it via pip:

pip install flask

Once installed, you can create a new Flask application by importing the Flask module:

from flask import Flask
app = Flask(__name__)
  • Step 1: Install Flask
  • Step 2: Create a new Python file
  • Step 3: Import Flask and initialize your app
Routing

Routing in Flask is simple. You define routes using the @app.route() decorator:

@app.route('/')
def hello_world():
    return 'Hello, World!'

Each route is associated with a function that gets called when the route is accessed.

  • Define a route using the decorator
  • Associate a function with the route
  • Return content from the function
Templates

Flask supports templating using Jinja2. This allows you to separate your HTML from Python code:

from flask import render_template
@app.route('/home')
def home():
    return render_template('home.html')

Create a template file in a templates directory and render it using the render_template function.

  • Store templates in a dedicated folder
  • Use Jinja2 syntax for dynamic content
  • Render templates from your view functions
Handling Requests

Flask makes it easy to handle different types of HTTP requests (GET, POST, etc.).

from flask import request
@app.route('/submit', methods=['POST'])
def submit():
    data = request.form['data']
    return f'You submitted: {data}'

Using the request object, you can access form data, query parameters, and more.

  • Use request to access incoming data
  • Specify allowed HTTP methods with the methods parameter
  • Process and return data from your routes
